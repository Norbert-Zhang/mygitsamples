# Git
## Git Bash Commands (wq => exits edit mode; clear => clear the screen)   

git init (init/create the current folder to a local repository)   
git status (show change status)   
git diff (show differences)   
git add . (add to stage)    
git commit -am "commit message" (commit to head)   
git log (show all commit infos)   
touch README.md (create new file README.md)   
touch .gitignore (create new file .gitignore)   

git branch (show all the branches)  
git branch -M main (change the master to main)  
git remote add origin https://github.com/Norbert-Zhang/es6.git (connect the remote repository)  
git push -u origin main (update to server)  

git clone https://github.com/Norbert-Zhang/es6.git (clone a local repository from the server and connect the server)  

git push (update to server)  
git pull (get from server)   

git branch (show all the branches)  
git branch newbranch (create a newbranch)   
git checkout newbranch (change the branch to newbranch)   
git push -u origin newbranch (create a newbranch in the server)   
git checkout main (change the branch to main)     
git merge newbranch (merge the newbranch to main)   
git branch -d newbranch (Deleting local branches)   
git push origin --delete newbranch (Deleting remote branches)
